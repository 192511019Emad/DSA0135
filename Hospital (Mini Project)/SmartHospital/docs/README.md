# AI Enhanced Smart Hospital Management System

Complete C++17 console project for smart patient management, AI-inspired risk prediction, emergency prioritization, and hospital bed resource management.

## Build Command

```bash
g++ -std=c++17 -Iinclude src/*.cpp -o SmartHospitalApp
```

## Run Command

```bash
./SmartHospitalApp
```

On Windows PowerShell:

```powershell
.\SmartHospitalApp.exe
```
